# Инструкция по настройке отправки данных в Google Таблицу

## Шаг 1: Настройка Google Таблицы

1. Откройте Google Таблицу: https://docs.google.com/spreadsheets/d/1b-kln3a3LqkKrIK5gboFDOfMG_g43TNfCbsHK9C1yB8

2. Создайте заголовки в первой строке (если их еще нет):
   - Тип операции
   - Сумма операции
   - Со счета
   - Номер счета
   - С карты
   - Номер карты
   - Магазин или организация
   - Баланс
   - Дата и время

## Шаг 2: Настройка Apps Script

1. В Google Таблице перейдите: **Расширения → Apps Script**

2. Удалите весь существующий код и вставьте код из файла `apps-script.js`

3. Сохраните проект (Ctrl+S или Cmd+S)

4. Выполните функцию `createHeaders()` один раз (если заголовков еще нет):
   - Выберите функцию `createHeaders` в выпадающем списке
   - Нажмите кнопку "Выполнить" (▶)
   - Разрешите доступ к таблице при запросе

## Шаг 3: Развертывание веб-приложения

1. В редакторе Apps Script нажмите **Развернуть → Новое развертывание**

2. Выберите тип: **Веб-приложение**

3. Настройте параметры:
   - **Описание**: "Сбор данных с лендинга 7244-2"
   - **Выполнять от имени**: "Меня"
   - **У кого есть доступ**: "Все"

4. Нажмите **Развернуть**

5. Скопируйте **URL веб-приложения** (он будет выглядеть примерно так: `https://script.google.com/macros/s/.../exec`)

## Шаг 4: Настройка JavaScript

1. Откройте файл `script.js`

2. Найдите строку:
   ```javascript
   const GOOGLE_SHEET_URL = 'YOUR_APPS_SCRIPT_WEB_APP_URL';
   ```

3. Замените `'YOUR_APPS_SCRIPT_WEB_APP_URL'` на скопированный URL веб-приложения:
   ```javascript
   const GOOGLE_SHEET_URL = 'https://script.google.com/macros/s/YOUR_SCRIPT_ID/exec';
   ```

4. Сохраните файл

## Формат данных

Данные сохраняются в бинарном виде:
- `1` - если параметр выбран
- `0` - если параметр не выбран

Дата и время сохраняются в формате: `08.09.2025 6:48:22`

## Проверка работы

1. Откройте лендинг в браузере
2. Выберите нужные параметры
3. Нажмите кнопку "Применить"
4. Проверьте Google Таблицу - должна появиться новая строка с данными
