# СМС без лишних слов

Лендинг для выбора шаблонов SMS-уведомлений о банковских операциях.

## Описание

Проект представляет собой мобильный лендинг, позволяющий пользователям выбрать один или несколько шаблонов SMS-сообщений, которые они будут получать при совершении операций по карте.

## Функционал

- Выбор одного или нескольких шаблонов SMS-сообщений
- Визуальная индикация выбранных шаблонов
- Кнопка применения выбранных настроек
- Адаптивный дизайн для мобильных устройств
- Плавные анимации и переходы

## Использование

1. Откройте `index.html` в браузере
2. Выберите один или несколько шаблонов SMS-сообщений
3. Нажмите кнопку "Применить" для сохранения настроек
